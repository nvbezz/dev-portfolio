Portafolio personal donde muestro mis proyectos, habilidades y experiencia como desarrollador.

## Stack
- Next.js 16 + React 19
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Vercel

## Correr localmente

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Variables de entorno

Copia `.env.example` a `.env.local` y completa los valores:

```bash
cp .env.example .env.local
```
